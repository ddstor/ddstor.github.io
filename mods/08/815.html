Урок 15. Тайминг

Домашнее задание:

минимум

  1. Напиши список дел, которые тебе нужно сделать сегодня. Возле каждого дела напиши время, которое тебе нужно для выполнения этого дела. И сделай одно дело.

100%

  1. Сделай все дела из списка по методу «сжатие-соревнование». После каждого пятого, десятого действия делай себе награду. Если сделаешь все дела из списка – отметь это особенно для себя и запиши этот день в список своих достижений.

Максимум

  1. Запиши видео-отзыв, как у тебя получилось побороть лень с помощью этого способа. Поделись им в отчете на платформе.
